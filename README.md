# PlatePal

PlatePal is a food discovery and meal planning application designed to connect food enthusiasts with diverse culinary experiences. The platform provides personalized restaurant recommendations, health-conscious meal planning, and a community-driven approach to discovering new dishes and restaurants.

## Features

- **Restaurant Discovery**: Browse and discover restaurants tailored to your preferences
- **Dish Exploration**: Explore detailed information about dishes at various restaurants
- **Health Tracking**: Track your dietary preferences and health goals
- **Meal Planning**: Create personalized meal plans based on your dietary needs
- **User Profiles**: Customize your preferences and view your activity
- **Favorites**: Save your favorite restaurants and dishes for quick access
- **Search**: Powerful search functionality to find restaurants and dishes

## Tech Stack

- **Frontend**: React with TypeScript
- **Backend**: RESTful API
- **Database**: Optimized data management for food and restaurant information

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/Abdulrehman0911/PlatePal.git
cd PlatePal
```

2. Install dependencies
```bash
npm install
# or
yarn install
```

3. Set up environment variables
```bash
cp .env.example .env.local
```

4. Start the development server
```bash
npm run dev
# or
yarn dev
```

## Project Structure

- `/src` - Main source code
- `/public` - Static assets
- `/components` - Reusable React components
- `/pages` - Page components and routing
- `/api` - API integration and services
